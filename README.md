## :computer: Projeto
Estudos de:

**1. Tipos Básicos:**
   - String
   - Number
   - Boolean
   - Array
   - Tupla
   - Enum
   - Any
   - Unknown
   - Null e Undefined
   - Object

**2. Multiplos tipos com Build Types:**
   - Void e Never
   - Type Alias, Union e Intersection
   - Type Assertions

**3. Orientação a objeto:**
   - Classes, Interfaces, Heranca e Modificadores de Acesso
